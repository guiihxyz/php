recomendações para dar nomes:
1 - Tente dar nomes claros e de fácil identificação
2 - Evite nomes muito curtos ou muito longos
3 - Defina um padrão e siga em todo o projeto
4 - Para variáveis, dê preferência a letras minúsculas
5 -Para constantes, dê preferência a letras maiúsculas
6 - Use camelCase para métodos e atributos, exemplo: nomeCompletoCliente
7 - Use SNAKE_CASE para nomear constantes, exemplo: TELEFONE_CONTATO_FORNECEDOR

Criar um Array:
Array Indexado:
$array = array("Maçã", "Banana", "Laranja");

Array Associativo:
$frutas = array("a" => "Maçã", "b" => "Banana", "c" => "Laranja");

Acessar Elementos do Array:
echo $array[0]; // Saída: Maçã
echo $frutas['b']; // Saída: Banana

Adicionar Elementos:
$array[] = "Pera"; // Adiciona 'Pera' ao final do array indexado
$frutas['d'] = "Abacaxi"; // Adiciona 'Abacaxi' ao array associativo

Contar Elementos:
echo count($array); // Retorna o número de elementos no array

Iterar por um Array:
foreach:
foreach($array as $valor) {
    echo $valor . ", ";
}
// Saída: Maçã, Banana, Laranja, Pera,

Verificar se um Elemento Existe no Array:
if (in_array("Maçã", $array)) {
    echo "Maçã está no array!";
}

Ordenar um Array:
sort($array); // Ordena um array indexado
asort($frutas); // Ordena um array associativo mantendo a associação com os valores

Remover Elementos do Array:
unset($array[1]); // Remove o elemento de índice 1 do array indexado
unset($frutas['b']); // Remove o elemento com a chave 'b' do array associativo
